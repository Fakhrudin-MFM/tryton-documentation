Server Installation
===================

The Tryton server provides the system's core functionality, amongst other
things it combines business logic, access control, and data from the database,
to respond to requests from the clients.

There are several different ways of installing the Tryton server software onto
your machine, and the right choice for you will depend on what software you
have available on your machine, and how you want to set Tryton up.
