    model
      A model represents a single concept or idea.  It is commonly used to
      allow things in the real world to be represented and stored in the
      system.

      .. hint::

          If you needed to store information about lots of different cars
          then a 'Car' model might be appropriate, each property that was
          important to you would be a :term:`field` and each car would be
          stored as a :term:`record`.
